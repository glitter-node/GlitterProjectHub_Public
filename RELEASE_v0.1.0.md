# Glitter Project Hub v0.1.0

Initial public release of Glitter Project Hub.

Glitter Project Hub is an experimental collaboration and archive platform built on top of Gnuboard7 with a Laravel-based application structure.

This public release focuses on the bundled extension architecture and operationally safe deployment structure.

## Included

- Bundled module architecture
- Bundled plugin architecture
- Bundled template architecture
- Runtime-safe activation lifecycle
- Laravel integrated application structure
- Archive-oriented collaboration flow
- Installer/runtime separation design
- Template activation workflow
- Language pack lifecycle structure

## Excluded

This public repository intentionally excludes:

- Operational runtime data
- Production environment configuration
- Deployment infrastructure
- Internal operational documents
- Seed content
- Active installed extensions/templates
- Secrets and environment-specific files

## Repository Structure

Public releases expose only bundled source packages through:

- modules/_bundled/
- plugins/_bundled/
- templates/_bundled/

Installed runtime copies are intentionally excluded from version control.

## Status

This project is currently experimental and under active development.
Internal structures and APIs may change without notice.
